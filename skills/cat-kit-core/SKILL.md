# @cat-kit/core

## 安装

```bash
npm add @cat-kit/core
```

## 查证 API

本技能目录下 [`generated/`](generated/) 中的 **`.d.ts`** 与 npm 包内 `dist` typings 一致。通常从 [`generated/index.d.ts`](generated/index.d.ts) 起读。

**运行环境**：通用（浏览器与 Node.js 均可，无内置 Node 专用 API 依赖）。
