# @cat-kit/vitepress-theme

VitePress 文档站主题（水墨丹青风格），通过 `defineConfig` 注册使用。

## 使用方式

```ts
import { defineConfig } from 'vitepress'
import catKitTheme from '@cat-kit/vitepress-theme'

export default defineConfig({ theme: catKitTheme })
```
