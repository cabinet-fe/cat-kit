# @cat-kit/vitepress-theme

## 安装

```bash
npm add @cat-kit/vitepress-theme vitepress vue
```

## 查证 API

[`generated/`](generated/) 下 **`.d.ts`** 与 npm `dist` typings 一致；入口 [`generated/index.d.ts`](generated/index.d.ts)。样式相关声明见 [`generated/style.css.d.ts`](generated/style.css.d.ts)。

**运行环境**：VitePress 2 文档站（浏览器 + 构建期 Node）；peer 依赖 `vitepress`、`vue`。
