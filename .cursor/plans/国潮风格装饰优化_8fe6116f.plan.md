# 国潮风格装饰优化计划

## 一、新增二十四节气组件

创建 [`docs/.vitepress/theme/components/SolarTerms.vue`](docs/.vitepress/theme/components/SolarTerms.vue)

### 设计方案

采用**书法卷轴**风格，在首页展示当前节气：

```javascript
┌─────────────────────────────┐
│   ░░░卷轴装饰边░░░           │
│                             │
│      「 立 春 」            │  ← 节气名称（篆书/楷书风格）
│                             │
│     ☘️ 相关图案              │  ← SVG 图案（如嫩芽、雪花、荷花等）
│                             │
│   2月4日 · 春季              │  ← 节气日期和季节
│                             │
└─────────────────────────────┘
```



### 功能特性

1. **自动节气计算**：根据当前日期判断节气（使用公历近似算法）
2. **节气专属图案**：24个节气各有对应的 SVG 图案（如立春-嫩芽、小暑-蝉、白露-露珠）
3. **季节配色**：四季对应不同的点缀色（春-青玉绿、夏-朱砂红、秋-鎏金、冬-靛蓝）
4. **入场动画**：卷轴展开效果

### 放置位置

在首页 `#home-hero-after` 插槽，与印章相对，位于左侧或中下位置，不遮挡主要内容。---

## 二、改进墨渍背景效果

重构 [`docs/.vitepress/theme/components/InkBackground.vue`](docs/.vitepress/theme/components/InkBackground.vue)

### 当前问题

- 墨渍形状过于规则，不够自然
- SVG 滤镜效果较为生硬
- 整体缺乏"渗透"和"晕染"质感

### 改进方案

采用**多层叠加 + 渐变遮罩**的方式模拟真实墨渍效果：

1. **基础墨渍层**：使用更有机的 SVG 形状（参考真实墨渍轮廓）
2. **晕染效果**：使用径向渐变 + blur 模拟墨汁边缘的渗透
3. **纸纹交互**：与宣纸纹理结合，使用 `mix-blend-mode` 增强纸墨交融感
4. **动态微变**：加入极微弱的动画，模拟墨汁缓慢渗透扩散

### 设计细节

```css
/* 核心改进思路 */
.ink-blot {
  /* 多层渐变叠加 */
  background: 
    radial-gradient(ellipse at 30% 40%, rgba(26,26,26,0.08) 0%, transparent 50%),
    radial-gradient(ellipse at 70% 60%, rgba(26,26,26,0.05) 0%, transparent 40%);
  
  /* 柔和边缘 */
  filter: blur(20px);
  
  /* 正片叠底模拟渗入纸张 */
  mix-blend-mode: multiply;
}
```
