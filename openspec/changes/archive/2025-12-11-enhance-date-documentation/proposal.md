# Change: 增强日期处理文档 - 添加交互式示例

## Why

当前 `@cat-kit/core` 日期处理文档只有静态代码块，缺乏交互式演示。用户难以直观感受 API 的实际效果（如日期格式化、计算、比较等），降低了文档的可用性和学习效率。

## What Changes

- 为日期处理文档（`docs/content/packages/core/date.md`）添加多个交互式 demo 组件
- 创建 Vue 组件展示以下核心功能：
  - **日期格式化**：实时展示不同格式模板的输出效果
  - **日期计算**：展示加减天数、月份、年份的结果
  - **日期比较与范围判断**：可视化日期差值和区间判断
  - **日期解析**：展示从字符串解析日期的效果
- 使用现有的 `:::demo` 容器语法嵌入示例到文档中

## Impact

- **受影响的文件**：
  - `docs/content/packages/core/date.md` - 需要添加 demo 容器引用
  - `docs/examples/core/` - 新目录，包含日期相关 demo 组件
- **受影响的规范**：无（纯文档改进，不影响 API）
- **用户体验提升**：用户可以直接在文档页面操作和观察日期工具的实际行为
