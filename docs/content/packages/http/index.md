# HTTP 请求包

`@cat-kit/http` 提供一个可扩展的 `HTTPClient`，支持 Browser、Node.js 和 Bun。它内置请求方法别名、插件链、错误类型、下载进度、自定义引擎，以及 `TokenPlugin`、`RetryPlugin`、`MethodOverridePlugin` 等常用扩展。
