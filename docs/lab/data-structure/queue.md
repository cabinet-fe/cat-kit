# 队列 Queue

队列是一种线性的数据结构, 其命名源自现实生活中的排队, 先排到的人先完事.

js的数组本身可以完全用来表示队列.

```ts
const queue: number[] = []

// 入队
queue.push(1)
queue.push(2)

// 出队
queue.shift()
queue.shift()
```

## 使用场景
- js的事件循环机制, 就是基于两个队列(宏任务队列, 微任务队列)
